Linux / macOS Deployment
1
Install Python dependencies
pip install websockets psutil
2
Extract agent script
Click to extract siem_agent.py to an accessible folder:
3
Edit line 14 — set your server IP
# Change to your AKD server IP: SIEM_SERVER = "ws://YOUR_IP:8765"
4
Run agent (sudo required for log access)
sudo python3 siem_agent.py
5
Optional: Run as systemd service
# /etc/systemd/system/akd-agent.service [Unit] Description=AKD Security Agent After=network.target [Service] ExecStart=/usr/bin/python3 /opt/siem_agent.py Restart=always [Install] WantedBy=multi-user.target
sudo systemctl enable --now akd-agent
